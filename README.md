# Country Comparison

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 11.1.1.

## This project was developed with

Angular, Angular Material, NgxCharts and REST Countries API

## Development server

Download project, open in editor, run `npm install` in the terminal. After instalation is complete, run `ng serve` for a dev server. Navigate to `http://localhost:4200/` in your browser.
